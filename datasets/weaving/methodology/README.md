# Weaving Methodology

This documentation describes the life cycle inventory (LCI) modelling and impact assessment for woven fabric production, covering all major loom technologies and yarn fineness variants.

| Field | Value |
| --- | --- |
| **Technology** | Weaving (shuttle, rapier, projectile, air-jet, water-jet, Jacquard) |
| **Functional unit** | 1 kg of woven fabric ready for wet processing |
| **Scope** | Warping, sizing, drawing-in, loom operation, inspection |
| **LCIA method** | EF 3.1 (16 indicators) |
