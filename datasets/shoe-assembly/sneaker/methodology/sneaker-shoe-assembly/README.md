# Sneaker Shoe Assembly Methodology

## Methodological specification of the sneaker shoe assembly inventory model

| Property | Value |
|---|---|
| **Functional unit** | 1 pair of footwear (assembly operations only) |
| **Assembly processes** | Lamination, Cutting, HF Welding, Screen Printing, Insole, Nosew, Stitching, Stock fit, Assembly |
| **Electricity scenarios** | Vietnam (VN) and Global average (GLO) |
| **Total datasets** | 2 (VN and GLO electricity scenarios) |
| **Background database** | ecoinvent 3.12, cut-off |
| **LCIA method** | EF v3.1 — all 16 impact indicators |

**📄 [Download full methodology (PDF)](sneaker-shoe-assembly.pdf)**

The PDF contains the complete methodology documentation, including modelling strategy, chemical input construction, proxy decisions, and material loss treatment (IOR and defect rates).
