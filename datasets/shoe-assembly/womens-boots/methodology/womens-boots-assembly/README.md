# Women's Boots Assembly — Methodology

Full methodology documentation is available in the PDF: **[womens-boots-assembly.pdf](womens-boots-assembly.pdf)**

## Summary

Women's boots assembly (PU/Microfiber Upper + TPR/EVA/Rubber Outsole) covering three sequential operations:

1. **Cutting** (IOR = 1.0382): Fabric cutting with oil paint printing for pattern marking
2. **Stitching** (IOR = 1.006): Upper stitching with 163N glue application
3. **Lasting** (IOR = 1.004): Final assembly using PU adhesive, BW553 primer, and surface cleaner

| Parameter | Value |
|-----------|-------|
| Functional unit | 1 kg assembled footwear (~1 pair) |
| Aggregated loss rate | 1.049 (= 1.0382 × 1.006 × 1.004) |
| Electricity scenarios | CN (China grid), GLO (global average), IT (Italy grid) |
| Data source | Primary data from a Wenzhou (China) shoe manufacturer |
| Background database | ecoinvent 3.12 (cut-off) |
