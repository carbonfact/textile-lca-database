# Shoe Assembly

> Lifecycle assessment datasets for footwear assembly operations, covering sneaker and women's boots manufacturing processes.

**5 datasets** across 2 assembly types | All 16 EF 3.1 impact indicators

## Assembly types

| Assembly type | Datasets | Electricity scenarios | Functional Unit | Description |
|--------------|----------|----------------------|-----------------|-------------|
| [Sneaker](sneaker/) | 2 | VN, GLO | 1 pair of footwear | Sneaker assembly with 9 sub-processes (lamination, cutting, HF welding, screen printing, insole, nosew, stitching, stock fit, final assembly) |
| [Women's Boots](womens-boots/) | 3 | CN, GLO, IT | 1 kg assembled footwear | Women's boots assembly (PU/Microfiber Upper + TPR/EVA/Rubber Outsole) with 3 sub-processes (cutting, stitching, lasting) |
