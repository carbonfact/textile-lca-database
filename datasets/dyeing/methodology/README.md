# Dyeing Methodology

This document describes the life cycle assessment methodology for textile dyeing processes, covering inventory development, impact assessment, and benchmarking across multiple technology variants and dye classes.

| Item | Detail |
| --- | --- |
| **Technology variants** | Exhaust (jet, airflow jet, softflow, beam, hank, package, front loader), Semi-continuous (CPB), Continuous (pad-steam, thermosol, range, rope/slasher) |
| **Functional unit** | 1 kg of dyed textile, cradle-to-gate (excluding pre-treatment) |
| **Scope** | Core dyeing, washing/rinsing, drying/tumble drying/padding |
