# Ring Spun Yarn Methodology

## Methodological specification of the ring-spun yarn inventory model

| Property | Value |
|---|---|
| **Functional unit** | 1 kg of finished yarn, at the spinning mill gate |
| **Fibre types** | Cotton, Acrylic/PAN, MMCF, Polyester, Polyamides, Elastane, Silk blends, Wool (worsted & woollen), various blends |
| **Dtex range** | 45--600 dtex |
| **Total datasets** | 875 individual process datasets |
