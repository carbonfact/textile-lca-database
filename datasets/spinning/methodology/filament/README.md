# Filament Spinning Methodology

## Methodological specification of the filament yarn (melt/dry/wet) spinning inventory model

| Property | Value |
|---|---|
| **Functional unit** | 1 kg of finished filament yarn at the spinning mill gate |
| **Process types** | Melt spinning (POY, FDY, DTY), Dry spinning, Wet spinning |
| **Material families** | Polyester, Polyamides, Acrylic/PAN, Elastane, Acetate, MMCF, Microfibers, Recycled PA 6.6 |
| **Dtex range** | 45--600 dtex (energy values constant across dtex) |
| **Total datasets** | 828 rows (18 process/material combinations x 46 dtex values) |
