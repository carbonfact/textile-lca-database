# Open End Rotor Methodology

## Methodological specification of the open-end rotor yarn inventory model

| Property | Value |
|---|---|
| **Functional unit** | 1 kg of finished OE yarn at the spinning mill gate |
| **Fibre types** | Cotton, CVC, PC, Cotton/man-made, Wool/man-made, Linen blends, Polyester, Poly blends, Acrylic/PAN, Polyamides, Polyamide blends, MMCF, Silk blends |
| **Dtex range** | 40--600 dtex |
