# Natural Rubber — STR Production Methodology

Full methodology documentation is available in the PDF: **[natural-rubber-str.pdf](natural-rubber-str.pdf)**

## Summary

Cradle-to-gate LCA of Standard Technical Rubber (STR) production, from rubber plantation through STR processing. The model has a 3-level nested activity structure:

1. **Field coagulum production** — Rubber plantation model with economic allocation (17.8% to field coagulum). Includes fertilizers, pesticides, wood burning, country-specific LUC and peatland drainage.
2. **STR processing** — Conversion of field coagulum into STR (2 kg coagulum → 1 kg STR). Includes electricity, diesel, LPG, water, wastewater, formic acid.
3. **Market for STR (GLO)** — Production-weighted mix of TH (46%), ID (26%), CI (15%), VN (12%).

| Parameter | Value |
|-----------|-------|
| Functional unit | 1 kg STR (Standard Technical Rubber) |
| System boundary | Cradle-to-gate (plantation through STR processing) |
| Background database | ecoinvent 3.12 (cut-off) |
| Allocation | Economic allocation for plantation co-products |
| Country variants | TH, ID, CI, VN |
| Key references | Birnbach et al. (2020), Jawjit et al. (2010) |
